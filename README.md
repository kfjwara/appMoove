# appMoove

iPhone向けオフライン動画プレイヤー PWA。Brave Playlist のオフライン再生の代替。

**アプリ:** https://kfjwara.github.io/appMoove/

## 使い方

1. iPhone の Safari で上記URLを開く
2. 共有メニュー →「ホーム画面に追加」
3. ホーム画面の Moove を起動 →「＋ 動画を取り込む」で mp4 等を選ぶ（「ファイル」アプリから選択可）
4. 以降は機内モードでも再生できる

## 仕組み（Phase 1）

- 動画は OPFS（アプリ専用ストレージ）に Worker 経由でチャンク書き込み（大容量対応）
- メタデータ（タイトル・再生位置・尺）は IndexedDB
- Service Worker がアプリ本体をキャッシュしてオフライン起動
- 再生位置の自動保存・レジューム・連続再生つき

### 注意

- Safari のタブとホーム画面アプリはストレージが別。動画は**ホーム画面アプリ側で取り込む**こと
- 保存上限は端末の空き容量に依存（概ね空きの6割程度）。空きが逼迫すると OS に消される可能性はある
